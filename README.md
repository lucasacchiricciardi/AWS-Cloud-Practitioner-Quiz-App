# AWS Cloud Practitioner Quiz App

A premium, static web application to test your AWS Cloud Practitioner knowledge.

## Features
- **Persistent Progress**: Your progress is saved automatically using Local Storage.
- **Instant Feedback**: Immediate validation of answers with explanations.
- **Responsive Design**: Works on Desktop and Mobile.
- **Premium UI**: Modern glassmorphism aesthetic.

## Deployment to Vercel

This project is ready for static deployment on Vercel.

1.  Push this folder to a GitHub repository.
2.  Import the repository in Vercel.
3.  Vercel will automatically detect the static `index.html`.
4.  Click **Deploy**.

## Local Development
Simply open `index.html` in your browser.
